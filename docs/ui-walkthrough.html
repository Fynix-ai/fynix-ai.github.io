Fynix UI Walkthrough

Assistant Chat View

Click on the Fynix-Copilot Icon or use the shortcut Cmd + Shift + 0 to open the sidebar, which contains the Assistant Chat View. This is where users interact with Fynix Copilot. The Assistant Chat View has four tabs: Chat, Guide, Settings, and History.

New Conversation: Click on the plus icon on the top-right of the Assistant Chat View to open a new conversation.

Chat Tab

In the Chat Tab, you have a welcome screen with quick commands to get started. At the bottom, you'll find the Chat Input Field. Type your message and press Enter to send it to Fynix Copilot and get a response.

Mentions (@)

Click on the @ symbol or type @ in the Chat Input Field to open a list of supported mentions. Mentions allow you to add more context to your interaction with Fynix Copilot.

Supported Mentions:

  • @File: Opens a list of files to choose from, allowing for questions and commands on that file.
  • @Code: Opens a list of functions in the repository for questions or commands on a specific function.
  • @Folder: Opens a list of folders in the repository to ask questions about a folder.
  • @Git Diff: Opens a list of Git diffs to select and add context to the chat.
  • @Workspace: Mentions the entire workspace for context.
  • @Web: Searches the query on the web.
  • @Docs: Opens a list of user-added documents to reference in the chat.
  • @Jira: Mentions Jira tickets.
  • @Thread: Adds context from previous conversations using Chat ID.
Image Attachment

Click on the attachment icon at the bottom of the Chat Input Field to open the file manager (Finder/Windows Explorer). Select an image to attach it to the Chat Input Field and ask questions about it.

Model Selection

Click on the model name in the Chat Input Field to open a list of models to select from. The selected model will be used to generate responses.

AI Review

Click on the Magic Wand icon at the bottom-left of the Assistant View or the top-right of the VS Code window to open the AI Review tab, or use the shortcut Cmd + Shift + R. AI Review provides a comprehensive review of code quality and potential bugs.

Fynix Diagram

Click on the "Flowchart" icon at the bottom of the Assistant Chat View or the top-right of the VS Code window to open the Fynix Diagram tab. This feature generates flowcharts and sequence diagrams based on your prompt or code.

Fynix Terminal

Click on the "Terminal Window" icon at the bottom of the Assistant Chat View or the top-right of the VS Code window to open the Fynix Terminal tab, or use the shortcut Cmd + Shift + L. The Fynix Terminal is a natural language terminal.

Guide Tab

Click on the Guide tab to quickly access instructions on how to use Fynix Copilot. It contains four dropdown sections: Overview, Slash Commands, Mentions, and Shortcuts. Use the search bar to find specific topics.

Settings Tab

The Settings tab allows you to customize your Fynix Copilot experience. It has five sub-sections: General Settings, Advanced Settings, Account Settings, Model Settings, and API Keys.

History Tab

The History Tab consists of all your previous conversations with Fynix. Users can search for a particular chat using keywords. Each conversation has a unique Chat ID, which can be used in a new conversation to extract context from a previous chat. Users can also delete conversations by clicking on the "bin" icon.

View and More Actions

Users can click on the three horizontal dots at the top of the tabs to view additional options, including:

  • Export Conversation: Export the current conversation into a new file, which will open in the code editor.
  • Logout: Log out of the current session.
  • Report Bugs: Open a pop-up window to report bugs to the Fynix Copilot team.

Code Interaction Tools

Fynix Copilot provides several tools for interacting with generated code:

  • Quick Reply: Hover over a paragraph in the generated code to click on the reply button for further questions.
  • Quick Copy: Hover over a paragraph to copy specific parts of the text.
  • Copy Whole Response: Click on the copy icon at the end of the response to copy all content.
  • Regenerate Using a Different Model: Use the regenerate button at the end of the response to get a new result.

Generated Code Options

For generated code snippets, users can:

  • Flowchart: Generate a diagram for the code using the "flowchart" icon.
  • Insert At Selection: Replace the selected code in the file with the generated code.
  • Insert In New File: Paste the generated code into a new file.
  • Insert In Terminal: Paste the code into a new terminal window.

Input Editing and Threading

Users can edit input commands or prompts by clicking the "edit" icon beside the input. Generating a new response from the edited input creates a new thread. Users can navigate between threads by clicking the left arrow for the previous response and the right arrow for the next response.

Feedback

Users can provide feedback on generated responses by clicking the thumbs-up or thumbs-down icons at the end of the response.
