Quickstart

Quickly learn to use Fynix in your Editor.

Introduction

Welcome to Fynix Copilot, your AI programming assistant designed to accelerate your coding productivity by offering:

  • Real-time code suggestions
  • Project scaffolding
  • Code completion
  • Code fixing
  • Code documentation
  • Cross-language translation
  • Real-time web search
  • AI review
  • Git difference analysis
  • Natural Language Terminal
  • Flowchart Diagram
Whether you’re a beginner or a seasoned developer, Fynix Copilot integrates seamlessly with your workflow, guiding you through the Software Development Lifecycle.

Prerequisites

Before you get started, ensure that you have the following:

  • Compatible IDE: Visual Studio Code (latest version)
  • Fynix Copilot Plugin: Install it from the VS Code Marketplace or search for Fynix in the VS Code extensions tab. Once installed, you will see the Fynix Copilot icon below the extensions icon on the left-hand side of VS Code.

Login: You’ll need to log into your Fynix Copilot account after installing the plugin. You can either sign up from scratch or log in using existing credentials (Google or Outlook).

Chat with Fynix Copilot

Once installed, you can begin interacting with Fynix Copilot right from your editor. Here’s how:

  1. Open Visual Studio Code and click on the Fynix Copilot icon (Shortcut: Cmd+Shift+0) in the activity bar.
  2. The Assistant chat view will appear. Type your questions or commands in the input field at the bottom of the chat view, and press enter to send.
  3. Fynix will respond interactively to your queries.
