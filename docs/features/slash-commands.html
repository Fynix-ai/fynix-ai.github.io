Type / followed by a command name and press Enter to select the command. Press Enter again to execute the command in the chat. Please note: You need to select a code block or file before using a slash command.

Slash Commands

1. Get Help with /help

Use /help to get a list of available slash commands and their brief descriptions.

2. Project Scaffolding with /new

Automatically create project scaffolding based on your input.

Example: Type /new web app in the chat to generate a basic directory structure and files for a web application. Click “Create Workspace” to finalize.

3. Code Improvement with /fix and /refactor

Fix errors or optimize code with these commands. Improve readability and performance effortlessly.

Example: Select the code to improve, type /fix, and press Enter. Similarly, use /refactor to optimize the selected code.

4. Cross-Language Translation with /translate

Convert code from one programming language to another.

Example: Select a code block, type /translate Python to C++, and the Copilot will handle the conversion.

5. Unit Test Generation with /test

Automate the creation of unit tests for your functions.

Example: Select a function, type /test, and Fynix will generate the unit tests.

6. Documentation with /doc

Generate docstrings for the selected code or file.

Example: Select a code block, type /doc, and send the command to generate documentation automatically.

7. Code Explanation with /explain

Get a line-by-line explanation of selected code.

Example: Type @index.html /explain to receive a detailed explanation of the file named index.html.

8. Complete the Code with /complete

Generate or complete code from a description or partially written code.

Example: Select the definition of Add_to_cart(), type /complete, and the Copilot will suggest the complete function implementation.

9. Visualize Code with /diagram

Generate a logical diagram of your code.

Example: Type @main.py /diagram to generate a diagram of the Python file main.py.
