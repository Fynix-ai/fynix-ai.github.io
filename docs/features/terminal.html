Use the Fynix Terminal to interact with your system using natural language. It converts natural language instructions directly into executable terminal commands, eliminating the need to memorize commands.

Steps to Use Fynix Terminal

  1. Click on the "Fynix Terminal" icon at the bottom of the Assistant Chat view or the top right of the VSCode window to open the Fynix Terminal tab.
  2. Write your instruction in natural language and press Enter to send it to the Fynix Terminal. The terminal will generate the corresponding command.
  3. Review the command and press Enter to execute it.

Note:

As best practices, save environment variables and directory structures in "Rules for AI" to provide Fynix Terminal with the necessary context.

Examples

1. Installing Requirements

If you have a file named requirements.txt containing a list of libraries for your project, type: 

install requirements

Fynix Terminal will generate:

$ pip install -r requirements.txt

Press Enter to execute the command.

2. Creating Requirements File

Instruction to Fynix Terminal:

create a new file named requirements.txt and write the list of dependencies required for the following project in the requirements.txt file. 
Project description: An e-commerce website using React for the frontend, Node.js for the backend, and MongoDB for the database. The website should include features like payment gateway integration, product recommendations, and user authentication.

Fynix Terminal might generate:

$ echo 'react 
react-dom
axios
redux
react-redux
react-router-dom
express
mongoose
jsonwebtoken
bcryptjs
stripe
passport
passport-local
passport-jwt
cors
body-parser
dotenv
nodemon' > requirements.txt && echo 'Created requirements.txt with the list of dependencies.'

3. Automated Workspace Setup

Scenario: A developer wants to set up a workspace for a new project. They add project setup steps in "Rules for AI":

  • Clone the repo: https://github.com/org/project-x.git
  • Create a Python virtual environment inside the project folder.
  • Install dependencies from requirements.txt.

They type:

Set up my workspace for Project X

Fynix Terminal generates:

git clone https://github.com/org/project-x.git && cd project-x && python -m venv env && source env/bin/activate && pip install -r requirements.txt && npm install
