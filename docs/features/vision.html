Click on the attachment icon at the bottom of the Chat Input Field to open the ‘Finder’ or ‘Windows Explorer’. Select an image of your choice and click Open. You can now interact with the image directly in the Fynix Copilot chat.

Example

Upload an inspiration of the UI that you want to create for your website, and type the following prompt:

Create a similar webpage in HTML.

Fynix Copilot will analyze the image and generate the HTML code for a webpage that closely matches your design inspiration.
