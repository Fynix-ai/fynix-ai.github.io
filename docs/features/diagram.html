Use Fynix Diagram to visualize code snippets, project structures, or create UML diagrams from prompts.

Steps to Use Fynix Diagram

  1. Click on the "Flowchart Icon" at the bottom of the Assistant Chat view or the top right of the VSCode window to open the Fynix Diagram tab.
  2. Type or paste your prompt/code in the input field of the Fynix Diagram tab and click on "Convert To Diagram".
  3. Fynix will generate a Flowchart/Sequence Diagram. You can:
    • Move the diagram by click-hold-drag
    • Zoom in and out
    • Download the diagram in supported formats

Note:

Diagrams are automatically generated when using the /explain and /doc commands. You can also use the /diagram command directly in the Assistant Chat view to generate diagrams.
