Mentions Features

1. Mention and Interact with a File

Use @file and press Enter to see the list of files in the entire workspace. Select a specific file and chat with it using natural language or slash commands. Directly use @ followed by the name of the file to mention and interact with it.

Use Cases:

  • Dependency Management Across Multiple Files: Type: Generate a list of dependencies based on the following files: @file1.filetype, @file2.filetype, etc. Then click on “Insert in new file” to generate a file like requirements.txt.
  • Contextual Debugging of Specific Files: Mention files like @main.py and @utils.py to analyze their relationship and identify potential issues like mismatched function parameters or missing imports.
  • Cross-File Code Refactoring: Mention files like @file1.py and @file2.py to refactor duplicate code into a new module like common_utils.py.

2. Mention and Interact with Specific Functions

Use @code and press Enter to see a list of functions in the workspace. Select a specific function and interact with it using natural language or slash commands.

3. Mention and Interact with a Folder

Use @folder and press Enter to see the list of folders in the workspace. Select a specific folder and interact with it using natural language or slash commands.

4. Get Suggestions on Git Diff

Use @Git to see a list of commits. Select one to view differences and get suggestions from Fynix. Use slash commands like /explain to get detailed explanations of Git differences.

5. Workspace Analysis with Mentions

Fynix Copilot offers context-aware suggestions based on your codebase. Mention @Workspace in the chat to analyze the entire workspace, ask questions, and receive tailored insights.

6. Get Answers from Web

Use @web to let Fynix browse the internet and provide relevant information. Example: @web list down features from the latest Python release.

7. Interactive Document Engagement with @docs

Engage with documents interactively. Use @docs to select a saved document or add a new one. Fynix will assist in discussions based on the document's content.

8. Jira Tickets

Mention Jira tickets directly. Configure your organization's Jira link and API key in settings. Use: @jira [ticket ID] [instructions] to interact with Jira tickets and resolve them in your IDE.

9. Auto Detect Links

Paste any link in the chat to interact with it. Example: Paste a documentation link of a library to ask questions or gain insights.
