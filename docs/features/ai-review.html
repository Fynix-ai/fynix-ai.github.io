AI Review

Home Docs Home Features Overview

Use the AI Review feature to analyze changes made to the current Git repository. Fynix provides suggestions to fix issues, improve performance, and catch bugs or errors before submitting your pull request. Issues are categorized by type (e.g., Bug, Coding Standard, Critical) for easy reference.

Steps to Use AI Review

  1. Click on the Magic Wand icon at the bottom left of the assistant view or the top right of the VSCode window to open the Fynix AI Review tab, or use the shortcut Cmd+Shift+R.
  2. (Optional) Provide custom review instructions for a focused review. Example: Focus on performance or Look for logical inconsistencies.
  3. Click the "Review Working State" button to see a list of suggestions.
  4. Click "Fix Issue" to let Fynix generate a new code snippet based on the suggestions from the AI Review.
  5. Select the code you want to replace and click the "Insert at Selection" option in the newly generated code snippet.
  6. Review the changes and click "Accept and Insert" or use the shortcut Cmd+Y to insert the new code at the selected position.
